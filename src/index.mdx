# RnB Extra Product Options

RnB Extra Product Options is a user friendly Extra Field Options for your Rental and Booking site.It works as an extension for RnB - WooCommerce plugin. Extra Product Options allows you to generate Extra Field on RnB . It comes with the following Fields:

### Extra Fields:
-   Text Field
-   Textarea Field
-   Number Field
-   Color Field
-   Checkbox Field
-   Radio Field
-   Select Field
-   Multi Select Field
-   Date & time Field
### *Requirements:

You must have the following plugin installed

[WooCommerce plugin](https://wordpress.org/plugins/woocommerce/)
