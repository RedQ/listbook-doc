# FAQ

## What is RnB extra product option? 

Answer: EPO is an extension for RnB -WooCommerce plugin.It helps you to create and generate extra custom field on RnB Rental Product.

## What Functionalities does EPO Provide?

Answer:EPO Helps to Generate extra field for RnB Product.

## What is the requirement for EPO to function properly?

Answer:You must have RnB- WooCommerce plugin installed. EPO work as an RnB extension in order to work you will be RnB WooCommerce plugin.

## How to setup EPO Fields?
`Inventory -> Settings -> Extra filed`
