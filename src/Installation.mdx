# Installation

After download the full package from CodeCanyon, you will have `rnb-extra-product-options.zip` file inside your main package. You can install it from WordPress back-end or
manually.

## Install from WordPress back-end:

```
1. Go to wordpress Dashboard -> plugins -> add new.
2. From here click upload plugins button, then you will find the plugin upload window.
3. Upload rnb-extra-product-options.zip then click install now button.
4. After successfully install the plugin then active it.
```
